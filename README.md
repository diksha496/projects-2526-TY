# 🌸 Women Self Help Group (SHG) Management System

## 📌 Project Overview
The Women Self Help Group (SHG) Management System is a web-based application designed to manage SHG activities efficiently. It helps in maintaining records of members, savings, loans, EMI, and bank loan applications.  

Traditional SHG systems rely on manual record-keeping, which is time-consuming and error-prone. This system digitizes all operations, improving accuracy, transparency, and efficiency.

---

## 🎯 Objectives
- To manage member information effectively  
- To maintain savings records of each member  
- To provide and track loans  
- To monitor EMI payments and defaulters  
- To generate reports for analysis  
- To reduce manual paperwork  

---

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **Database:** SQLite  
- **Tools:** VS Code / PyCharm, Web Browser  

---

## 📊 System Modules

### 👤 Member Management
- Add, view, edit, and delete members  
- Store personal details  

### 💰 Savings Management
- Record member savings  
- View saving history  

### 💳 Loan Management
- Provide loans to members  
- Store loan details and interest  

### 📅 EMI Management
- Track current EMI  
- Track paid EMI  
- Identify defaulters  

### 🏦 Bank Loan Module
- Apply for bank loan  
- Upload required documents  
- Track application status  

### 📈 Reports
- Generate member reports  
- Savings reports  
- Loan and EMI reports
